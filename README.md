
# Infinity Runner

Projeto desenvolvido durante o 8º módulo do Curso de Desenvolvimento Unity da CSJ Academy. Infinity Runner é um jogo onde o jogador a cada jogada tentara ir mas longe do que a ultima vez. 


## Aprendizados

- Parallax
- Herança
- Colisões
- Interface

## Funcionalidades

- Atirar
- Pular
